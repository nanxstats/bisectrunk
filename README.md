## Installation

### Prerequisites

Install Rust:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Install C compiler and linker:

```bash
sudo apt-get update && sudo apt-get install -y build-essential
```

### Install bisectrunk

From crates.io (stable release):

```bash
cargo install bisectrunk
```

From GitHub (latest development version):

```bash
cargo install --git https://github.com/nanxstats/bisectrunk.git
```

**Note**: If `cargo` or `bisectrunk` is not found immediately after installation,
restart your shell.
